# Product Analytics Dashboards (Tableau)

This repository contains a collection of interactive dashboards developed to analyze user behavior, business growth, and product performance.

### 1. Cohort Analysis: Registrations & Conversion Rates
This is my primary project focused on user journey and monetization.
 **Objective:** Analyze how different registration cohorts convert to trial and paid subscriptions.
 **Key Metrics:** Conversion Rate (CR), Retention, User Growth.
 **Technical Skills:** LOD Expressions (FIXED), Cohort Heatmaps, Funnel Visualization.
 [View on Tableau Public](https://public.tableau.com/views/CohortAnalysisRegistrationsConversionRates/CohortAnalysisRegistrationsConversionRates?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

### 2. Business Growth Overview 2022-2023
A high-level executive dashboard showing global business performance.
 **Focus:** Revenue by location, transaction distribution, and ARPU trends.
 **Insights:** Identified key growth regions and seasonality in user spending.
 [View on Tableau Public](https://public.tableau.com/views/BusinessGrowthOverview2022-2023/BusinessGrowthOverview2022-2023?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

### 3. Revenue Analysis Dashboard
A detailed breakdown of monthly recurring revenue and sales dynamics.
 **Focus:** New MRR, revenue growth %, and monthly payment trends.
 **Technical Skills:** Time-series analysis, interactive filters, and trend lines.
 [View on Tableau Public](https://public.tableau.com/views/RevenueAnalysisDashboard_17735833849130/RevenueAnalysisDashboard_?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link_)

### 4. Game Activity Analysis
Analyzing player engagement and progression for a mobile game.
 **Focus:** Level funnel (identifying where players drop off) and activity heatmaps.
 **Insights:** Optimized the game balance by finding bottlenecks in early levels.
 [View on Tableau Public](https://public.tableau.com/views/GameActivityAnalysis_17735730121740/GameActivityAnalysis?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

All dashboards are fully interactive. Click the links above to explore the data filters and visualizations.
